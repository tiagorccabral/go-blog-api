## Simple Blog API

---

#### About

This is a simple Blog API made with the Go programming language.

### How to run

### Setup
  - Clone Repo
  - Update .env file

#### Using Docker
This project uses docker.
So to get it running simply run the command:
```
docker-compose up
```

By default, the server should now be running on:

`http://localhost:8000/`

To shut it down:
```
docker-compose down
```
